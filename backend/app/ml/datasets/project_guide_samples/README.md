# FinGenie Project Guide Sample Datasets

This folder contains presentation-ready sample datasets in INR.

Files:
- `sample_transactions_inr.csv`: transaction-level sample data (income + expenses)
- `sample_budgets_inr.csv`: sample monthly category budgets
- `sample_user_preferences_inr.csv`: sample user-level app preferences
- `sample_monthly_series_inr.csv`: aggregated monthly series for analytics/forecast demos
- `sample_mcc_reference_subset.csv`: small MCC mapping subset for category explanation
- `sample_cpi_subset.csv`: small CPI subset for inflation-aware insights

All files are CSV and can be opened in Excel/Google Sheets.
